# Parking Lot - Row with Maximum Occupied Spaces

This repository contains a Java solution for finding the row in a parking lot matrix that has the maximum number of occupied spaces.

## Problem
A parking lot is represented as a matrix of size R × C where:

0 = Empty parking space  
1 = Occupied parking space

The goal is to find the row index that contains the maximum number of 1s.

## Example Input

3  
3  
010  
110  
111  

## Output

3

## How to Run

Compile:
javac Main.java

Run:
java Main

## Language
Java
